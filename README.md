## Running Locally

### Prerequisites
- Node.js (16+ recommended)
- npm or yarn

### Installation

1. Install dependencies:
```bash
npm install
```

2. Set up environment variables:
```bash
cp .env.example .env.local
# Edit .env.local and add your GEMINI_API_KEY if needed
```

3. Run the development server:
```bash
npm run dev
```

4. Open http://localhost:5173 in your browser

### Building for Production

```bash
npm run build
```

The built files will be in the `dist/` directory.

## Design Philosophy

This implementation demonstrates:
- **Modern Architecture**: Component-based React patterns
- **Type Safety**: TypeScript for maintainable code
- **Developer Experience**: Vite for fast builds and HMR
- **Modularity**: Separated concerns with dedicated components
- **Scalability**: Architecture that can easily add new features

## Key Design Decisions

1. **React + TypeScript**: Industry-standard stack for type-safe UI development
2. **Vite**: Modern build tool replacing webpack for faster development
3. **Tailwind CSS**: Utility-first CSS for rapid styling
4. **Component Separation**: Each major section as its own component
5. **Modular Structure**: Easy to extend and maintain
